START

Rule of the Game

Press any key to start the game. Once the game is start the machine will generate a random colour and the player has to remember the color and press it. On successful pressing the level will increase and the machine will continue to add new colour at each level. The player is expected to remember the whole sequence from level 1 to proceed to next level. Once the player fails to press the correct sequence the game will be over.
